# Pre-requisites

CCS v3.1 was primarily designed to operate on Windows XP. To achieve this, you’ll need to set up a virtual machine on your PC. [[1](https://www.dsprelated.com/thread/11025/tms320c6713-projects)] suggested that [VMware](https://www.vmware.com/products/workstation-player.html) is the most suitable choice for this scenario. Additionally, you’ll need to download a [Windows XP iso](https://archive.org/details/xp-sp-3-pt-pt-progr) image.

# Installation
